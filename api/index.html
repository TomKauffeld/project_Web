The API for the blog website

current version : v1
api link : https://pubflare.ovh/school/blog/api/latest
All responses will be in json format
User
 Get an user
 Create an user
 Login

  • User

    a User is made up with an id, an username, and an adminLvL
    adminLvL :
     0 : user (can post comments)
     1 : administrator (can post posts)
     2 : super admin (can control who gets which adminLvL)

    • Get an user

      get an user via an GET request to /user/{id} with {id} being the id of the user
      exemple :
      request :
      GET https://pubflare.ovh/school/blog/api/latest/user/g5RpsA
      response :
      {
       "status": "OK",
       "user":
       {
        "id": "g5RpsA",
        "username": "tom",
        "adminLvL": 1
       }
      }

    • Create an user

      to create an user send a POST request over a secure https connection with inside the body the parametres username and password set
      exemple :
      request :
      POST https://pubflare.ovh/school/blog/api/latest/user
      username="tom"&password="123456789"
       response if succesfull :
      {
       "status": "OK",
       "user":
       {
        "id": "g5RpsA",
        "username": "tom",
        "adminLvL": 1
       },
       "token":
       {
        "a": "g5RpsA",
        "b": 1557766952654126,
        "c": "YIEUFLfebyfzfbskif"
        "d": "GEEOIGOZGDJliegodls"
       }
      }
       response if not succesfull :
      {
       "status": "ERROR",
       "error": "THE USERNAME IS ALREADY TAKEN"
      }

    • Login

      to login send a POST request over a secure https connection to /user/login with inside the body the parametres username and password set
      exemple :
      request :
      POST https://pubflare.ovh/school/blog/api/latest/user/login
      username="tom"&password="123456789"
       response if succesfull :
      {
       "status": "OK",
       "loggedIn": true,
       "user":
       {
        "id": "g5RpsA",
        "username": "tom",
        "adminLvL": 1
       },
       "token":
       {
        "a": "g5RpsA",
        "b": 1557766952654126,
        "c": "YIEUFLfebyfzfbskif"
        "d": "GEEOIGOZGDJliegodls"
       }
      }
       response if not succesfull :
      {
       "status": "OK",
       "loggedIn": false
      }
