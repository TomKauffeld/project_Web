• user

    • GET /user

      gets all users
      optional parameters :
      limit : how many users you want to get
      offset : how many users you want to skip
      exemple :
      request https://pubflare.ovh/school/blog/api/latest/user?limit=2&offset=0
      response {"status":"OK","lenght":2,"users":["abcd","efgh"],"version":"v1"}

    • GET /user/{id}

      gets an user
      parameter :
      id : the id of the user you want to get
      exemple :
      request https://pubflare.ovh/school/blog/api/latest/user/abcd
      response {"status":"OK","user":{"id":"abcd","username":"test","adminLvL":0},"version":"v1"}

    • GET /user/{id}/comments

      get all comments made by a user
      parameter :
      id : the id of the user you want to get the comments from
      exemple :
      request https://pubflare.ovh/school/blog/api/latest/user/abcd/comments
      response {"status":"OK","lenght":2,"comments":["htje","gesge"],"version":"v1"}

    • GET /user/{id}/posts

      get all posts made by a user
      parameter :
      id : the id of the user you want to get the posts from
      exemple :
      request https://pubflare.ovh/school/blog/api/latest/user/abcd/posts
      response {"status":"OK","lenght":2,"posts":["fesfe","gregr"],"version":"v1"}

    • POST /user

      creates a new user
      parameters (must be in a json format inside of the request body) :
      username : the username of the user to create (at least 3 characters)
      password : the password of the user to create (at least 6 characters, different than the username)
      exemple :
      request url https://pubflare.ovh/school/blog/api/latest/user
      body {"username":"test","password":"thisIsAPassword"}
      response {"status":"OK","token":{"id":"abcd","b":157952,"c":"gesg","d":"fesf"},"user":{"id":"abcd","username":"test","adminLvL":0},"version":"v1"}

    • PATCH /user/{id}

      changes the adminLvL of an user
      parameters :
      id : the id of the user to change the adminLvL (must be in the url)
      token : your token (you must have an adminLvL of 2 in order to execute this comment), (must be in a json format inside the request body)
      adminLvL : the new adminLvL (must be either 0, 1 or 2), (must be in a json format inside the request body)
      exemple :
      request url https://pubflare.ovh/school/blog/api/latest/user/abcd
      body {"token":{"id":"abcd","b":157952,"c":"gesg","d":"fesf"},"adminLvL":2}
      response {"status":"OK","version":"v1"}

  • category

    • GET /category

      gets all categories
      exemple :
      request https://pubflare.ovh/school/blog/api/latest/category
      response {"status":"OK","lenght":2,"categories":["aecd","eegh"],"version":"v1"}

    • GET /category/{id}

      gets a category
      parameter :
      id : the id of the category you want to get
      exemple :
      request https://pubflare.ovh/school/blog/api/latest/category/aecd
      response {"status":"OK","user":{"id":"aecd","name":"test","description":"this is a test category"},"version":"v1"}

    • GET /category/{id}/posts

      get all posts made that have a specific category
      parameter :
      id : the id of the category you want to get the posts from
      exemple :
      request https://pubflare.ovh/school/blog/api/latest/category/aecd/posts
      response {"status":"OK","lenght":2,"posts":["fesfe","gregr"],"version":"v1"}
    • POST /category : create a category
    • PATCH /category/{id} : update the name

  • post

    • GET /post[?limit={limit}[&offset={offset}]] : get all posts
    • GET /post/{id} : get a post
    • GET /post/{id}/comments : get the comments on the post
    • POST /post : create a post

  • comment

    • GET /comment[?limit={limit}[&offset={offset}]] : get all comments
    • GET /comment/{id} : get a comment
    • POST /comment : create a comment

  • auth

    • POST /auth/validate : validate a token
    • POST /auth/login : login a user and get a token
